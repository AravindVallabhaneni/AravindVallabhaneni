## 🚀 Featured Projects

<details>
<summary><b>📄 AI Resume Analyzer & Job Matcher</b></summary>

<br/>

> AI-powered full stack application for resume analysis and job role recommendation, combining NLP-based parsing with intelligent matching.

<div align="center">

| Attribute | Details |
|-----------|---------|
| **Stack** | React, Node.js, Express.js, MongoDB, OpenAI API |
| **Core Features** | Resume parsing, skill extraction, keyword matching |
| **AI Integration** | NLP and OpenAI API for intelligent recommendations |
| **Frontend** | Responsive interfaces for resume upload and results |
| **Backend** | REST APIs with MongoDB for efficient data management |

</div>

**How It Works:**
- Users upload resumes which are parsed to extract skills, experience, and keywords
- NLP and OpenAI API integration analyze resume content against job requirements
- Keyword matching algorithm recommends suitable job roles based on extracted skills
- MongoDB stores parsed resume data and matching results for quick retrieval
- Responsive React frontend displays analysis results and job recommendations

</details>
